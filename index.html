Hint: Think about using display: flex; for the layout. You can use nav, section, and footer HTML elements.
Answer: <!DOCTYPE html>

}

}

nav, footer {
  background-color: #333;
  color: white;
  padding: 1rem;
}

section {
  flex: 1;
  padding: 20px;
}
</style>
</head>
<body>
<div class="container">
<nav>This is the navigation bar</nav>
<section>This is section 1</section>
<section>This is section 2</section>
<footer>This is the footer</footer>
</div>
</body>
</html>

Explanation: This code uses Flexbox to structure the layout of a webpage with a navigation bar, two sections, and a footer. Flexbox allows for flexible and responsive layouts, which are essential for modern web design.

Hint: A leap year is divisible by 4 but not by 100, unless it is divisible by 400.
Answer: function isLeapYear(year) {
  if ((year % 4 === 0 && year % 100 !== 0) || (year % 400 === 0)) {
    return true;
  } else {
    return false;
  }
}
console.log(isLeapYear(2024)); // true
console.log(isLeapYear(2023)); // false

Explanation: This JavaScript function checks if a given year is a leap year. It first checks whether the year is divisible by 4 but not divisible by 100, except for when it is divisible by 400. This logic ensures correct leap year detection.

Hint: Use dot notation or bracket notation to access values in a JSON object.
Answer: let jsonData = {
  "name": "John",
  "age": 30,
  "city": "New York"
};

// Accessing data using dot notation
let name = jsonData.name;
console.log(name); // John

// Accessing data using bracket notation
let age = jsonData["age"];
console.log(age); // 30

Explanation: In this example, JSON data is stored in a JavaScript object. Dot notation (jsonData.name) or bracket notation (jsonData["age"]) is used to access values. JSON manipulation is essential when dealing with APIs and dynamic data.
