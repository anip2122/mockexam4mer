Hint: Use the <input type="email"> for email validation and required attribute to make fields mandatory.
Answer: <form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required><br><br>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required><br><br>
  <input type="submit" value="Submit">
</form>

Explanation: This code demonstrates a basic HTML form with input fields for name and email. The email field uses the type="email" attribute to ensure proper email validation. Additionally, the required attribute ensures that both fields must be filled in before submission.

Hint: Convert the string to an array using split(), reverse it with reverse(), and join it back to a string with join().
Answer: function reverseString(str) {
  return str.split('').reverse().join('');
}

console.log(reverseString("hello")); // Output: "olleh"

Explanation: This JavaScript function splits the string into an array of characters, reverses the array, and then joins it back into a string. This method allows the string to be reversed efficiently using built-in array methods.
