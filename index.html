Front-End Web Development Mock Exam

Page 1 - HTML Questions

  1. Write HTML code to create a form that includes an input for name, email, and a submit button.

    Use the <form>, <input>, and <button> tags.
    <form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name"><br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br>
    <button type="submit">Submit</button>
    </form>

  2. List and describe four different types of web hosting services.

    Think about shared, VPS, dedicated, and cloud hosting.
    1. **Shared Hosting**: Multiple websites share the same server resources.
    2. **VPS (Virtual Private Server) Hosting**: A virtual machine with dedicated resources, isolated from other websites.
    3. **Dedicated Hosting**: A physical server exclusively used by one website.
    4. **Cloud Hosting**: Resources come from multiple servers, ensuring high availability and scalability.

  3. Create the HTML to embed a video with a subtitle track. The video file is "example.mp4" and the subtitle is "subtitles.vtt".

    Use the <video> and <track> tags.
    <video controls>
    <source src="example.mp4" type="video/mp4">
    <track src="subtitles.vtt" kind="subtitles" srclang="en" label="English">
    </video>

Page 2 - CSS Questions

  1. Write CSS code to center an element horizontally using Flexbox.

    Use `display: flex` and `justify-content: center`.
    .container {
      display: flex;
      justify-content: center;
    }

  2. Explain the CSS box model and how padding and margin differ.

    Focus on content, padding, border, and margin.
    The **CSS box model** consists of:
    1. **Content**: The actual content of the element.
    2. **Padding**: Space between the content and the border. It increases the size of the element.
    3. **Border**: The line surrounding the padding (if set).
    4. **Margin**: Space outside the border, pushing elements away from each other. It does not affect the element's size.

  3. Write the CSS to change the background color of all even list items in an unordered list.

    Use the `nth-child` selector.
    ul li:nth-child(even) {
      background-color: lightgray;
    }

Page 3 - JavaScript Questions

  1. Write a JavaScript function that calculates the sum of two numbers.

    Use a function with two parameters and return the sum.
    function sum(a, b) {
      return a + b;
    }

  2. Explain the difference between local storage and session storage in JavaScript.

    Think about data persistence across sessions.
    **Local storage** stores data with no expiration date, allowing it to persist across browser sessions.
    **Session storage** stores data only for the duration of the browser session, meaning it is deleted when the session ends.

  3. Write JavaScript code to set and delete a cookie.

    Use `document.cookie` to manipulate cookies.
    To set a cookie:
    document.cookie = "username=JohnDoe; expires=Fri, 31 Dec 2024 12:00:00 UTC; path=/";

    To delete a cookie:
    document.cookie = "username=JohnDoe; expires=Thu, 01 Jan 1970 00:00:00 UTC; path=/";
