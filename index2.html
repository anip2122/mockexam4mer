Page 4 - HTML Concepts and Forms

  1. Write the HTML code for a form that allows a user to input their username, password, and age. Include appropriate input types and a submit button.

    Use <input type="password"> for the password field and <input type="number"> for the age.
    <form>
    <label for="username">Username:</label>
    <input type="text" id="username" name="username" required><br>
    <label for="password">Password:</label>
    <input type="password" id="password" name="password" required><br>
    <label for="age">Age:</label>
    <input type="number" id="age" name="age" required><br>
    <button type="submit">Submit</button>
    </form>

  2. What is the purpose of the <fieldset> and <legend> elements in HTML forms?

    These elements are used to group form elements.
    The <fieldset> element is used to group related elements in a form, and the <legend> element provides a caption for the <fieldset>. This improves form structure and accessibility.

Page 5 - CSS Box Model and Layout

  1. Describe the CSS box model and its components. How does padding affect the total size of an element?

    The box model includes content, padding, border, and margin.
    The **CSS box model** includes:
    1. **Content**: The actual content inside the element.
    2. **Padding**: Space between the content and the border, which increases the size of the element.
    3. **Border**: The line surrounding the padding (optional).
    4. **Margin**: Space outside the border, separating the element from others.

  2. Write the CSS code to create a 2-column layout using the `float` property. The left column should be 30% wide, and the right column should be 70% wide.

    Use `float: left;` and width percentages.
    .left-column {
      width: 30%;
      float: left;
    }

    .right-column {
      width: 70%;
      float: left;
    }

    .clearfix::after {
      content: "";
      display: table;
      clear: both;
    }

Page 6 - JavaScript Events and Storage

  1. Write a JavaScript function that adds an event listener to a button that alerts "Button Clicked" when the button is clicked.

    Use `addEventListener` to handle the event.
    document.getElementById("myButton").addEventListener("click", function() {
      alert("Button Clicked!");
    });

  2. Explain the difference between local storage and session storage in JavaScript. How would you store and retrieve a user's name using local storage?

    Focus on persistence across sessions and methods like `localStorage.setItem` and `localStorage.getItem`.
    **Local storage** persists data even after the browser is closed, whereas **session storage** is cleared when the session ends.

    Storing a name in local storage:
    localStorage.setItem("username", "JohnDoe");

    Retrieving the stored name:
    var name = localStorage.getItem("username");

  3. Write a JavaScript function to set and delete a cookie with a specified name and value.

    Use `document.cookie` for cookie manipulation.
    Setting a cookie:
    document.cookie = "username=JohnDoe; expires=Fri, 31 Dec 2024 12:00:00 UTC; path=/";

    Deleting a cookie:
    document.cookie = "username=; expires=Thu, 01 Jan 1970 00:00:00 UTC; path=/";
